Git - Это облако которое помогает сохранять свой код не теряя и не забывая
ее.Он очень удобен для больших проэктов с большим количеством участников

Github - Это социальная платформа, где ты можешь делится своим кодом.
При этом смотрящие твой код могут: дать тебе совет, подписаться, добавить
в друзья.
